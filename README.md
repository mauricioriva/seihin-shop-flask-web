Seihin Shop
===========

##  Installation
[Flask](https://flask.palletsprojects.com/en/2.0.x/installation/)

    $ pip install Flask

[Flask Mail](https://pythonhosted.org/Flask-Mail/)

    $ pip install Flask-Mail

## Set Env

    $ export FLASK_APP=SeihinShop
    $ export FLASK_ENV=development

## Create Database
    
    $ flask init-db

## Run the server

    $ flask run
